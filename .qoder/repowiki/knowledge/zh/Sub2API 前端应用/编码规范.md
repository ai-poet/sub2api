- 所有 API 模块均导出为命名常量（如 `authAPI`, `keysAPI`），并在 `src/api/index.ts` 中统一聚合导出。
- Pinia Store 采用 Composition API (`defineStore` with setup syntax) 编写，返回包含 state、computed 和 actions 的对象。
- 视图组件（Views）通过懒加载方式引入，路由定义中广泛使用 `component: () => import(...)` 以优化首屏性能。
- 业务逻辑复用通过 `composables` 目录下的 `useXxx` 函数实现，遵循 Vue 3 组合式函数的命名与组织规范。
