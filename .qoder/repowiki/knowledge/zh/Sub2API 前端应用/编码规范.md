- 所有异步 API 调用均通过 `src/api/client.ts` 中的 Axios 实例发起，并由拦截器统一处理 JWT 令牌附加与过期自动刷新逻辑。
- 视图层文件按角色权限严格划分为 `views/user`、`views/admin` 和 `views/auth` 目录，并通过路由守卫 `requiresAdmin` 元数据进行访问控制。
- 复杂业务逻辑（如 OAuth 流程、表格加载、防抖搜索）被抽取为 `composables/useXxx.ts` 组合式函数，以实现逻辑复用与测试隔离。
- UI 组件遵循原子化设计，通用基础组件存放于 `components/common`，业务特定组件存放于对应功能目录下，并配套编写 Vitest 单元测试。
