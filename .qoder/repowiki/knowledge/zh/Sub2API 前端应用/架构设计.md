- **技术栈与构建**：采用 Vue 3 + TypeScript + Vite 架构，使用 Pinia 进行状态管理，Vue Router 处理路由，Tailwind CSS 负责样式。
- **分层结构**：`src/api` 封装 Axios 客户端并处理 JWT 自动刷新；`src/stores` 管理全局认证与应用状态；`src/views` 按业务域（user/admin/auth）划分页面；`src/components` 复用 UI 组件；`src/composables` 提取逻辑钩子。
- **国际化与安全**：通过 `vue-i18n` 实现中英双语支持，并在请求拦截器中注入语言标识；针对 CSP 环境优化了 i18n 编译模式。
- **部署集成**：构建产物直接输出至后端 `backend/internal/web/dist` 目录，开发环境通过 Vite 插件注入后端公开配置以消除闪烁。
