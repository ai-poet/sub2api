- **技术栈与构建**：采用 Vue 3 + TypeScript + Vite 架构，使用 Pinia 进行状态管理，Vue Router 处理路由，Tailwind CSS 负责样式。
- **模块化结构**：`src/api` 封装 Axios 客户端及业务接口（区分用户端与管理端）；`src/stores` 集中管理认证、应用配置等全局状态；`src/views` 按角色（user/admin/auth）划分页面视图；`src/components` 沉淀通用 UI 组件与业务模块组件。
- **核心机制**：在 `main.ts` 中完成 Pinia、Router、i18n 的初始化与主题注入；`router/index.ts` 实现基于角色的访问控制（RBAC）导航守卫；`api/client.ts` 通过拦截器统一处理 JWT 令牌刷新、错误重定向及请求上下文注入。
